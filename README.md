# muTube – YouTube on muOS

**muTube** is my lightweight YouTube player for **muOS**, built for smooth video playback on low-power gaming devices. It lets you watch videos, download them for offline use, and control playback easily—all in a simple interface designed for handheld devices.

---

## Features
- Watch YouTube videos directly on muOS
--- 

## Planned Features
- Download videos for offline viewing
- Optimizations
- Basic playback controls for easy navigation  

---

## Why muTube?
I built muTube to experiment with media playback on muOS and create my own take on a lightweight YouTube experience. While projects like CTupe inspired me, muTube is my personal implementation for my workflow and device setup.
